HACKATHON CONTEXT Problem Statement: Automation of Formwork Kitting &
BoQ Optimization Using Data Science

NirmanAI addresses inefficiencies in: - Manual Bill of Quantities
(BoQ) - Excess inventory & capital blockage - Poor repetition planning -
Kit mismatch delays - Static planning in dynamic projects

------------------------------------------------------------------------

THE REAL PROBLEM

It’s 8:30 AM on site. The concrete pour is scheduled for 10 AM.

The site engineer recalculates the BoQ manually. A panel size mismatch
is discovered. Carpenters wait. Inventory lies unused in the yard.

Another delay. More capital blocked.

Formwork contributes 7–10% of total project cost, yet planning remains
manual and reactive.

------------------------------------------------------------------------

OUR SOLUTION: NIRMANAI

1.  Parametric Master Kits Using K-Means clustering, similar column
    dimensions are grouped into standardized master kits.

2.  Repetition Intelligence Maximizes kit reuse across floors to reduce
    duplication.

3.  Dynamic BoQ Engine Automatically recalculates BoQ when geometry
    changes.

4.  Just-In-Time Kitting Delivers only required components for the next
    pour cycle.

5.  Digital Feedback Loop Tracks usage, wear & lifecycle for continuous
    optimization.

------------------------------------------------------------------------

SYSTEM WORKFLOW
